# Investment-Behavior-Analytics   
**Overview**
A Power BI project analyzing investor psychology and financial patterns. This dashboard visualizes how demographics, risk appetite, and market trends drive investment decisions.

**Key Features**
Segment Analysis: Comparing Conservative vs. Aggressive investors.
Market Sentiment: Tracking buy/sell behavior during volatility.
Asset Allocation: Visualizing diversification across Stocks, Bonds, and Mutual Funds.
Retention Metrics: Analyzing churn and long-term holding patterns.

 **Tech Stack**
Power BI: Visualization & Star Schema modeling.
DAX: Metrics for ROI, Retention, and Portfolio Variance.
Power Query: Data cleaning and ETL.
